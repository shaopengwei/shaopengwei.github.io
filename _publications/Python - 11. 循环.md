# 循环

### 1. 循环是什么
重复执行相同代码。有两种循环方式：for 和 while

### 2. for 循环
语法：
for 变量 in 范围：
    循环语句块

### 3. while 循环
语法：
while 循环条件:
    循环语句块

### 4. break 和 continue
break 跳出所有循环
continue 跳出当前循环
